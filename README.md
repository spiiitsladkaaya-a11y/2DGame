# Neon Drift

**Neon Drift** — это компактный 2D indie-прототип на Unity с упором на *game feel*: плавное управление, процедурные визуальные эффекты, частицы, screen shake, процедурно сгенерированные звуки, фоновая музыка, современный UI, система очков, рестарт игры и сохранение рекорда.

---

## ▶️ Как запустить

1. Откройте проект в **Unity 2022.3 LTS** или новее.
2. Откройте сцену `Assets/Scenes/Main.unity`.
3. Нажмите **Play**.

Сцена намеренно сделана минималистичной.
`GameBootstrap` создаёт весь игровой прототип во время запуска, благодаря чему проект остаётся лёгким и удобным для изучения.

---

## 🎮 Управление

* Передвижение — `WASD` или стрелки.
* Удерживайте левую кнопку мыши, чтобы двигаться в сторону курсора.
* Нажмите `Space` после Game Over для рестарта.

---

## ✅ Реализованные функции

* Движение игрока с инерцией, ограничением камеры, наклоном, squash-эффектами и динамическим trail.
* Система спавна объектов:

  * collectible signal shards,
  * hazardous static blooms.
* Система очков и постепенное увеличение сложности.
* Экран Game Over:

  * плавное появление,
  * отображение финального счёта,
  * кнопка рестарта,
  * рестарт по клавише.
* Сохранение рекорда через `PlayerPrefs`.
* Процедурно генерируемые звуки:

  * подбор предметов,
  * столкновения,
  * UI,
  * Game Over.
* Процедурная фоновая музыка.
* Частицы, glow-эффекты, expanding rings, screen shake, hover-анимации кнопок и parallax background.
* Все спрайты генерируются во время выполнения — проект не использует внешние ассеты.
